# curso-vue

Curso VueJs

## Instalando o Vue CLI globalmente

```
npm i -g @vue/cli

```

## Criando projeto

```
vue create 'nome-do-projeto'

```

## Gerar build de produção

```
npm run build

```

## Gerar build Eletron

```
npm run electron:build

```

## Listar arquivos presets

```
cat .vuerc

```

## Adicionar plugin

```
vue add electron-builder

```

```
vue add vuetify

```

## Executando Electron

```
npm run electron:serve

```
